IF SOMETHING BREAKS
================================================================================

WhatsApp not working?
  → Check Twilio webhook URL is correct and uses HTTPS
  → Run: vercel logs --tail to see errors
  → Verify phone format is +1234567890

Deployment failing?
  → Check environment variables are correct
  → Ensure API keys don't have extra spaces
  → Check Node.js version: vercel env list

API 500 errors?
  → Check logs: vercel logs --tail
  → Look for [v0] debug messages
  → Verify all env variables are set
